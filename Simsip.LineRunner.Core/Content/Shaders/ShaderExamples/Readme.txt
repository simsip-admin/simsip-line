ARM Mali Shader Library README v1.0
===================================================================================

Files and directories contained in this directory (ShaderExamples):

 - src                  : Contains sepatrate directories for various shader effects

 - data                 : Data required (uniform and attributevalues) to render the 
                          shader effects in Mali Shader Development Studio

 - images               : Contains required images to render the effect of shaders 
                          in Mali Shader Development Studio

 - shaders.shaderconfig : A shader configuration file that contains pre-made
                          shader effects. This is required to render the shader effects
			  in Mali Shader Development Studio. This file specifies the
			  values of attributes and uniforms used in shader sources

 - Readme.txt           : Readme file for this directory

This directory contains following ESSL shader examples in src directory:

   AlphaBlend       
   blank            
   Brick Shader     
   BumpAndEnvMapping
   bumpmapping       
   DiscardColor      
   edging            
   environmentMapping
   Fog       
   Gooch     
   lighting  
   Mandlebrot
   normalViewer   
   raspberryripple
   redsquare      
   TexCoordView
   textures
   ToonShading
   VertexPosView
   vertexripple

More detailed information about the usage can be found in the 
Mali GPU Shader Library User Guide, found in part number
GX814-DC-08001.
